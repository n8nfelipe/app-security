# Roadmap

## Fase 1

- Gerar live ISO inicial.
- Fechar lista minima de pacotes.
- Aplicar hardening base no chroot.
- Validar boot em BIOS e UEFI.

## Fase 2

- Refinar identidade visual.
- Reduzir consumo de RAM e tempo de boot.
- Adicionar testes automatizados de boot e login.
- Definir politica de atualizacoes.

## Fase 3

- Integrar instalador grafico.
- Implementar opcao de disco criptografado.
- Publicar imagens assinadas.
- Construir repositorio proprio para overrides.

## Fase 4

- Perfil workstation e perfil server minimal.
- Telemetria opt-in localmente documentada, se fizer sentido.
- Guia de hardening adicional para ambientes corporativos.
