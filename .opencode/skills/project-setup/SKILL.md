name: project-setup
description: Configura ambiente Node.js + React automaticamente + python + php

instructions:
- se for javascript verificar se Node.js está instalado
- se for javascript validar presença de package.json
- criar arquivo .env se não existir (usar .env.example se disponível)
- se for javascript rodar npm run build se existir
- exibir erros caso falhe
- se for python verificar se tem o python instalado
- se for python validar a presença do pyproject.toml
- se for php verificar se tem o php instalado
- se for php verificar se composer esta instalado
- se for php validar a presença do composer.json
